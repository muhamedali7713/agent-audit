# Analysis Layer

This directory contains the analysis artifacts used by the paper-support dataset.

Included here:

- `summary.json`: headline corpus metrics
- `publication_report.md`: human-readable summary report
- `canonical_class_summary.json`: clustered issue-class counts
- `category_summary.json`: category-level summaries
- `cohort_comparison.json`: curated vs broad cohort comparisons
- `correlations.json`: key Spearman associations
- `findings_concentration.json`: concentration / Pareto metrics
- `surface_summary.json`: instruction-surface prevalence summary
- `surface_effects.json`: surface-presence effects
- `activity_covariates.json`: metadata coverage / maintenance summary
- `activity_effects.json`: activity-bucket summaries
- `repo_metrics_sanitized.json`: sanitized repo-level metrics keyed by `repo_key`
- `repo_analysis_509.{csv,jsonl}`: flattened joined repo-level table for the full 509-repo corpus
- `corpus_project_findings_sanitized.json`: sanitized corpus-wide `scan-project` output
